# Project Memory

## Core
Elite Live Trading Room for Infinox LATAM community. Red/crimson branding hsl(0,85%,50%). Deep black bg hsl(0,0%,4%).
Poppins font (headings+body). Maru Cyber Futuristic theme. Supabase Cloud backend. SSO integration with Infinox portal.

## Memories
- [Branding](mem://design/branding) — Red/crimson accent color, Infinox-inspired, Elite Live Trading Room name
- [Auth flow](mem://features/auth) — Email+password registration with display_name, SSO edge function for Infinox portal
- [Chat system](mem://features/chat) — Real-time Supabase channels + messages, persistent history, 9 seeded channels
- [Design style](mem://design/style) — Maru Cyber Futuristic theme, grid patterns, cyber-lines, rounded-full buttons, uppercase headings
