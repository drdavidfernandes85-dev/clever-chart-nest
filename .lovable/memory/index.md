# Project Memory

## Core
Elite Live Trading Room for Infinox LATAM community. Yellow branding hsl(45,100%,50%). Deep black bg hsl(0,0%,4%).
Poppins font (headings+body). Maru Cyber Futuristic theme. Supabase Cloud backend. SSO integration with Infinox portal.

- [Profile page](mem://features/profile) — /profile route with avatar upload (chat-attachments bucket) and display name editing
