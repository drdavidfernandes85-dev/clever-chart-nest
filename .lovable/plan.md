
## Chatroom Improvements Plan

### 🎨 Visual & Layout
1. **Avatar images** — Show profile photos (with fallback initials) instead of just initials
2. **Message grouping** — Collapse consecutive messages from the same user (only show avatar/name on the first one)
3. **Date separators** — Show "Today", "Yesterday", or date headers between message groups
4. **Typing indicators** — Show "User is typing..." animation at the bottom

### 💬 Messaging Features
5. **Emoji reactions** — Let users react to messages with emojis (👍, ❤️, 😂, etc.) like Discord/Slack
6. **Reply/quote** — Reply to a specific message with a quoted preview above your reply
7. **Message timestamps on hover** — Show full date/time on hover instead of always visible
8. **Edit & delete own messages** — Right-click or hover menu to edit/delete

### 📱 UX & Interaction
9. **Online users sidebar** — Show who's currently in the channel with online/offline status
10. **Unread badges** — Show unread message count on each channel in the sidebar
11. **Mobile-friendly sidebar** — Hamburger menu to toggle the channel list on mobile
12. **Scroll-to-bottom button** — Floating button when scrolled up, with new message count
13. **@mentions with autocomplete** — Type `@` to mention users with a dropdown picker

### 🔔 Engagement
14. **Browser notifications** — Notify users of new messages when the tab is in the background
15. **Sound effects** — Subtle notification sound for new messages
16. **Pinned messages** — Allow admins to pin important messages to the top of a channel

### Priority recommendation
Start with **items 2, 3, 6, 8, 11, 12** — these give the biggest UX lift with moderate effort.
